How to flash ESP32 with ESP32FREE Firmware and start using it
1. Connect your ESP32 to the PC
Connect your ESP32 device to the PC via USB.
! Important: Use the best USB-cable for the enough power supply.
2. Detect the COM-port number
Make sure that your PC detects esp32 and a new COM-port appears in the system when it's connected. Remember the port number.

! What if the system can't recognize connected esp32?
This means that the system does not have suitable drivers.
Seach for the drivers using the usb chip name on the ESP32 board.

Here are some known types:
CP210x usb to uart bridge: download driver here.
CH340 chip: download driver here.
3. Set the COM port number in Upload.cmd
Unpack the downloaded ESP32FREE.zip archive.
Open the file "Upload.cmd" in the Notepad.
Find the "COM5" text and replace "5" with your esp32 COM-port number (i.e. 3 for COM3).
Save the file.
4. Execute Upload.cmd
Double-click on the Upload.cmd to execute it.
Read the text on the screen. Find the button Boot on the ESP32 board and set the finger (or pencil) on it. It will take about 1 minute to flash the device.
When you are ready press any key.
Release the Boot button only when the device flashing stops.
Press any key once again to close the CMD window.
5. Connect to the ESP32 web-server
Reboot the esp32 after flashing.
Afer this start seaching for the wi-fi networks.
The esp32 Wi-Fi Access Point name will starts with ESP32_Dev...".
If you use the smartphone for connection it will be better to stop the Mobile internet on it (E, H+, 5G...).
Connect to the Wi-Fi Access Point "ESP32_Dev...".
Set-up ESP32FREE wi-fi
Go to the http://192.168.4.1
Connect your ESP32 as a client in your wi-fi network (or leave it as an access point).
Set the appropriate wi-fi mode settings on the Wi-Fi Settings page.
In wi-fi clent mode it will be better to set up the IPs manually, i.e.:
IP: 192.168.1.117
Mask: 255.255.255.0
Gateway: 192.168.1.1
Then press "Save and restart" button.
Set-up your device using ESP32FREE

After esp32 restarts, go to the ESP Settings page:
navigate to the (for example) http://192.168.1.117
and open the ESP Settings web-page in Menu.
- Add and set up your sensors and outputs.
- Create your variables on the Expressions panel.
- "Commutate" sensors and outputs adding Rules.
You'll see the sensors live values if the pins pointed correctly.
Monitor your device via the AutoDashboard
Open the Home page in the menu.
! Note: Visualisation available only if your esp32 in wi-fi client mode and your wi-fi network connected to the internet.

Use this auto dashboard page for monitor your device's sensors.
